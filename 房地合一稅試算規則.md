# 房地合一稅 2.0 試算規則

> 給未來的我（or 下一個 Claude）：這份是「在屋主面前手機操作」的版本，不要寫得太自動化、不要把屋主一定要選的欄位（自住、調職非自願）藏起來變成 AI 自己決定。**讓賞哥邊問屋主邊勾**才是正確流程。

---

## 0. 第一條鐵律：用 2.0 不要混到 1.0

**110 年 7 月 1 日（含）以後** 簽約成交（移轉登記日）的案子 → **全部走 2.0**。
舊制 1.0 範例 PDF 只用來對照欄位名稱，**稅率/費用率/持有期間級距完全不一樣**，不要混。

| 項目 | 舊制 1.0 | 新制 2.0（**我們用這個**）|
|---|---|---|
| 適用日 | 105/1/1 ~ 110/6/30 | 110/7/1 ~ 至今 |
| 持有 1 年內 | 45% | — |
| 持有 1 ~ 2 年 | 35% | 同 ≤ 2 年 → 45% |
| 持有 2 ~ 5 年 | — | **35%** |
| 持有 5 ~ 10 年 | 20% | **20%** |
| 持有 10 ~ 20 年 | 15% | **15%** |
| 持有 20 年以上 | 15% | **15%**（無級距變化）|
| 未提示證明的可減除費用 | 成交價 × **5%** | 成交價 × **3%**，**上限 30 萬** |
| 自住免稅額 | 400 萬 | 400 萬（不變）|
| 自住稅率 | 10% | 10%（不變）|

⚠️ **2.0 把「未提示費用率」從 5% 砍到 3% 並加 30 萬上限**。屋主自己估算時最常套錯這條。

## 1. 從謄本自動帶入的欄位（house-profile.html 已解析）

這幾欄不用手填，直接從建物/土地謄本所有權部抓：

| 欄位 | 謄本來源 | 申報書對應 |
|---|---|---|
| **所有權人姓名** | 所有權部 → 登記名義人 | 納稅義務人 (原所有權人) → 姓名 |
| **取得日期** | 所有權部 → 登記日期 | 「取得日期」欄（持有期間 RR 起算）|
| **取得原因** | 所有權部 → 登記原因 | 「取得原因」欄（轉成代號 a/b/c/d/e/f/h/z）|
| 房屋稅籍 / 段小段建號 | 建物標示部 | 房屋稅籍編號 / 段小段建號 |
| 土地坐落 | 土地謄本標示部 | 土地坐落（鄉鎮市區/段/小段/地號）|

### 登記原因 → 代號對照（謄本中文 → 申報書代號）

```
買賣           → a
拍賣 / 強制執行 → b
繼承           → c
遺贈           → d
贈與           → e（一般贈與）
夫妻贈與        → f（配偶贈與，需另填配偶原始取得原因）
廢止徵收        → h
其他（信託歸屬、判決移轉、共有物分割、第一次登記…）→ z
```

⚠️ **代號 g 在 2.0 已不用**（舊制是「自地自建」），自地自建在 2.0 改成案件類型勾選。

⚠️ **f 配偶贈與**：取得日期要填「配偶間第 1 次相互贈與前的**配偶原始取得日期**」，不是這次的登記日期。**這條超容易錯**，遇到 f 一定要追問屋主配偶當年怎麼買的。

⚠️ **c 繼承 / d 受遺贈 / h 廢止徵收**：取得日期可能不是登記日期（要看說明八），先抓登記日期當預設值並標「⚠️ 請確認」。

## 2. 屋主面前要問的手填欄位

按申報書順序，從上到下：

### 2.1 案件類型（勾選，多選）

```
□ 一般案件
   □ 房屋、土地（最常用）
   □ 房屋使用權
   □ 預售屋及其坐落基地
   □ 股份或出資額（特殊案）

□ 符合自住房地減免優惠條件 ← 屋主自己決定，要問！
□ 自住房地

□ 重購案件
   □ 先購後售
   □ 先售後購

□ 獨資或合夥組織交易案件（個人通常不用勾）

□ 因調職或非自願因素交易 ← 重要！稅率變 20%
□ 參與都市更新、危老重建 ← 重要！稅率變 20%
□ 自地自建
□ 合建分售
□ 合建分成
□ 合建分屋
□ 信託

□ 申請核發交易損失證明
```

UI 建議：折疊預設只顯示「一般案件 / 自住 / 重購 / 調職非自願 / 都更危老」5 個常見勾選，其他案件類型放「進階」按鈕展開。

### 2.2 納稅義務人居住身分（勾選）

```
□ 居住者（年滿 183 天，本國籍多半是）
□ 非居住者（外國買家 / 海外台商）
□ 國籍：中華民國 / 其他____
```

非居住者 → 直接 35% (>2年) / 45% (≤2年) **無分級**，沒有自住優惠。

### 2.3 金額類（最重要，也最容易算錯）

| 欄位 | 縮寫 | 說明 | 預設帶入 |
|---|---|---|---|
| 成交價額 | **AA** | 這次賣多少 | 從 house-profile 的「成交總價」帶入，可改 |
| 可減除成本 | **ZA** | 附表二合計 | 子表展開填 |
| 可減除移轉費用 | **ZB** | 附表三合計 | 子表展開填 |

成交價如果建物與土地沒有分開，可以**只填一欄合計**（附表一備註）。

## 3. 計算公式（縮寫對照 + 順序）

```
步驟 1：交易所得額
  AA 成交價額 - ZA 可減除成本 - ZB 可減除移轉費用 = GI 交易所得額（或損失）

步驟 2：課稅所得
  GI - ZC 前3年內交易損失 - 土地漲價總數額 = GL 課稅所得
  ※ GL < 0 → 填 0
  ※ 土地漲價總數額：土地稅法第30條第1項，公告土地現值計算（從土地增值稅單抓）

步驟 3a：適用自住優惠
  (GL - DM 自住免稅額 400 萬) × 10% = AF 應納稅額
  ※ GL ≤ 400 萬 → AF = 0
  ※ 應納稅額計算到「元」為止，尾數不滿 1 元無條件捨去

步驟 3b：不適用自住
  GL × 稅率 = AF 應納稅額

步驟 4：本次應自行繳納
  AF - ZD 重購自住房地扣抵稅額 = AH 本次應自行繳納稅額
```

### 縮寫總表（看到 AA/ZA/GI 不要慌）

| 縮寫 | 中文 | 出處 |
|---|---|---|
| AA | 成交價額 | 主表 |
| ZA | 可減除成本 | 附表二合計 |
| ZB | 可減除移轉費用 | 附表三合計 |
| GI | 交易所得額（或損失）| AA-ZA-ZB |
| ZC | 前3年內交易損失 | 附表四合計 |
| GL | 課稅所得 | GI-ZC-土地漲價總數額 |
| DM | 自住免稅額 | 固定 400 萬 |
| AF | 應納稅額 | GL × 稅率 |
| ZD | 重購扣抵稅額 | 附表五合計 |
| AH | 本次應自行繳納稅額 | AF-ZD |
| RR | 持有期間 | 取得日 → 交易日 |

## 4. 稅率表（2.0，**這個是現在用的**）

### 4.1 境內居住者

| 情形 | 持有期間 | 稅率 |
|---|---|---|
| **自住** | 不分（持有≥6年才符合自住資格）| **超過 400 萬部分 10%** |
| 非自住 | ≤ 2 年 | **45%** |
| 非自住 | > 2 ~ ≤ 5 年 | **35%** |
| 非自住 | > 5 ~ ≤ 10 年 | **20%** |
| 非自住 | > 10 年 | **15%** |
| **調職、非自願因素**（財政部公告原因）| ≤ 5 年 | **20%** |
| **自建或合建**（自有土地與營利事業合作，土地取得起 5 年內完成銷售）| 5 年內 | **20%** |
| **都市更新、危老重建**（第 1 次移轉，依條例參與）| ≤ 5 年 | **20%** |

### 4.2 非居住者（無分級、無自住優惠）

| 持有期間 | 稅率 |
|---|---|
| ≤ 2 年 | **45%** |
| > 2 年 | **35%** |

## 5. 自住房地優惠資格（屋主面前要問清楚）

**5 個條件全中才算自住**：

1. 出售房屋、土地辦竣**所有權移轉登記日**起算前 6 年內，個人或其配偶、未成年子女**設有戶籍**
2. 交易前 6 年內，**無出租、供營業或執行業務**使用
3. 個人與其配偶及未成年子女於交易前 6 年內**未曾適用本款規定**（一輩子能用一次喔，這條最容易踩雷）
4. 課稅所得超過 400 萬部分才課 10%
5. 房屋、土地必須**同時**符合以上條件

**屋主問答話術建議**（手機 UI 上做成 5 個 yes/no）：
1. 「您或配偶/未成年子女在這個房子的戶籍滿 6 年了嗎？」
2. 「這 6 年內有沒有出租或拿來做生意？」
3. 「您（含配偶/未成年子女）以前賣房有用過自住優惠嗎？」（**只要用過一次就不能再用**）
4. 「這個房子建物與土地是同一人所有嗎？」（如果建物和土地不同人，自住資格分別判斷）

5 題全 No / 全符合 → 顯示「✅ 適用自住優惠，稅率 10%（超過 400 萬部分）」
任一題不符 → 顯示「❌ 不適用自住，回到一般稅率」並列出原因

## 6. 附表二（ZA 可減除成本）—— 屋主常常少算的部分

```
ZA = ZA1 + ZA2 + ZA3
```

### ZA1：原始取得成本

**出價取得**（買賣、拍賣）：
- 當初買進的成交價 + 必要費用

**繼承、受贈取得**：
- 繼承/受贈日的房屋評定現值 + 公告土地現值
- × 消費者物價指數調整（依交易日的物價總指數）

**必要費用**（取得時的）：
- 契稅
- 印花稅
- 代書費
- 規費
- 仲介費（買進時付的）
- 公證費
- 增置、改良、修繕費（**這條超多人忘記！**裝潢、加蓋、廚房翻新、衛浴改造都算，要有發票）

### ZA2：改良土地費用

地方稅稽徵機關核准減除的改良土地費用（從土地增值稅繳款書「改良土地費用」欄抄）。

### ZA3（罕見）

申報書沒明列細項，留 0 即可。

## 7. 附表三（ZB 可減除移轉費用）—— **2.0 改了，注意 3% / 30 萬**

```
ZB = max(ZB1 + ZB3, ZB2 + ZB3)
```

### ZB1：已提示證明文件

**出售時付的**：
- 仲介費（屋主賣這次付給我們的）
- 代書費（過戶代書費）
- 規費（謄本費、登記規費）

⚠️ **2.0 沒有「清潔費」**了（1.0 有）。

### ZB2：未提示證明文件

```
ZB2 = min(成交價額 × 3%, 300,000)
```

⚠️ **2.0 是 3% 不是 5%！而且封頂 30 萬！**
舊制 1.0 是 5% 沒上限，賞哥要記得，1 億成交價 1.0 算 500 萬未提示費用，2.0 只算 30 萬，差 470 萬。

### ZB3：土地增值稅可減除部分

申報移轉現值**超過**公告土地現值的部分對應的土地增值稅。
> 土地增值稅本來是按公告現值計算，如果申報現值高於公告現值，超出部分繳的土增稅可以列為移轉費用。
> 從土地增值稅繳款書算。

### ZB 取較高者邏輯

實作時兩個都算，取大者：
```js
const zb = Math.max(zb1 + zb3, zb2 + zb3);
```

## 8. 附表四（ZC 前 3 年交易損失）

3 年內賣過其他房地有損失（GI < 0）且**已申報核定**的，可以拿來抵這次。
要有國稅局核定文號（檔案編號）。

實作上做成「+ 新增前次損失」按鈕，每筆填：
- 核定日期
- 檔案編號
- 交易日期
- 核定交易損失金額

通常屋主沒有，預設 0。

---

## 9. 附表五（ZD 重購自住扣抵）

只有勾「重購案件」才需要填。

```
條件：出售自住房地後 2 年內重購自住房地，或重購自住房地後 2 年內出售原房地

(1) = 重購價額
(2) = 出售價額
AF = 出售自住房地的應納稅額（從步驟 3 算來）

(1) ≥ (2) → ZD = AF（全額退）
(1) < (2) → ZD = AF × (1)/(2)（按比例退）
```

⚠️ 重購扣抵後 5 年內不得改作自住以外之用、不得移轉，否則會被追繳。

---

## 10. 土地漲價總數額（步驟 2 用）

不要自己算，**直接從土地增值稅繳款書的「土地漲價總數額」欄抄**。

> 公式：(申報移轉現值 - 前次移轉現值 × 物價指數調整) - 改良土地費用 - 工程受益費 - 其他可減項目
> 但這已經被地政事務所算過了，繳款書直接看數字就好。

---

## 11. 持有期間 RR 計算

```
RR = 交易日 - 取得日
```

- 取得日：謄本登記日期（除特殊案 c/d/f/h，另查說明七、八）
- 交易日：所有權移轉登記日（**不是簽約日**！）
- 持有期間以「年」計算，超過 1 日不滿 1 年算進該年

實作時用 dayjs / Date：
```js
const rrYears = (txnDate - acquireDate) / (365.25 * 86400 * 1000);
```

級距判斷：
```js
if (rrYears <= 2) bracket = '≤2年';
else if (rrYears <= 5) bracket = '2-5年';
else if (rrYears <= 10) bracket = '5-10年';
else bracket = '>10年';
```

## 13. 容易踩雷的地方（給未來的 Claude）

1. **2.0 vs 1.0 混用**：寫程式時稅率表硬碼，**只用 2.0**。1.0 案子幾乎絕跡了，要是真碰到舊案，提示屋主「這是舊制，請另外算」。
2. **未提示費用率寫成 5%**：是 **3%，且封頂 30 萬**。
3. **持有期間用簽約日算**：要用**移轉登記日**。
4. **f 配偶贈與取得日填錯**：要追到「配偶第 1 次相互贈與前的原始取得日」。
5. **自住優惠 6 年一輩子一次**：屋主常忘記以前賣房用過。
6. **GL < 0 沒設成 0**：課稅所得不能負，要 clamp 0。
7. **AF 尾數沒捨去**：規定「不滿 1 元無條件捨去」。
8. **重購扣抵的 (1)(2) 算錯**：(1)=重購價額，(2)=出售價額。
9. **建物與土地同人才同時自住**：分屬不同人時資格分判。
10. **非居住者沒有自住優惠也沒有分級**：勾居住身分時要 lock 掉自住勾選。
