## B. 手機版 HTML 匯出（A 案延伸）

### 🟢 2026-04-24 完工（6 commit：dfccbde → 3e21861）
詳見 `project_mobile_html.md`（memory）和 `手機版HTML匯出_規格.md`

### 交付功能
- 一鍵按鈕「📱 儲存成手機版（2 份）」
- 同時產出「**專員版.html**」（含聯絡）和「**無名版.html**」（隱去聯絡）
- 響應式：`@media (max-width:768px)` 手機單欄 / 桌面雙欄
- 位置圖 Google Static Maps base64 + **外層 `<a>` 點圖跳 Google Maps app**
- 地籍圖 SVG → Canvas 2x → PNG base64
- 離線可看（所有圖片 base64 嵌入）
- price-bar 5 指標手機下 grid 3 欄（2 行 3+2）
- `table-layout: fixed` + `word-break: break-word` 防 iPhone Safari 表格溢位

### 順便改的
- 兩個 Excel 按鈕檔名統一 `{案件名稱}-{物件地址}-{專員名}.xlsx`

### 關鍵決策
- **clone `.a4-card` DOM vs 手工模板**：選 clone，A4 版面改動自動跟上
- **檔名簡化**：原規格 `{案件}_專員版.html` 太長，賞哥改要純「專員版.html」

## C. 雲端化 + AI 建議書

### 🟡 規劃確認中（2026-04-24）
詳見 `專案藍圖_雲端化與AI建議書.md`

### 三階段路線圖
- **階段 1**：Google Drive 存檔 + 白名單控管（6-8 hr）
- **階段 2**：AI 建議書生成（8-12 hr）
- **階段 3**：白名單管理頁（條件性，人 > 5 才做，2-3 hr）

### 等賞哥回答 4 件事才能開工階段 1
1. 夥伴 Gmail 清單（至少賞哥自己）
2. UI 觸發方式：A4 右上按鈕 / 頁面頂部工作區 / 獨立管理頁
3. AI 選 Gemini（已付費代發 shang-gemini-proxy 有額度）還是試 Claude
4. 建議書格式：HTML + PDF / Word / Excel？

### 關鍵技術決策（已確認）
- 用個人 Gmail + 白名單（不用 Workspace 付費）
- Google Drive `drive.file` scope：天然隱私隔離
- 只存結構化資料，不存謄本 PDF
